# I. Introduction

- Vortex is a browser extension that acts as a personal AI academic advisor
- Designed for students in the City University of New York (CUNY)
- Goal: simplify academic planning with real-time, personalized guidance

# II. How Vortex Works

- Connects securely to student accounts after login
- Integrates with tools like DegreeWorks and Schedule Builder
- Collects key academic data:
- Completed courses
- Remaining requirements
- GPA
- Major and concentration
- Credit standing
- Uses this data to provide advice through a conversational interface

# III. Core Features

# A. Degree Progress Awareness

- Tracks academic progress
- Identifies completed and remaining requirements
- Recommends courses based on prerequisites and degree needs

# B. Stay-on-Track Guidance

- Detects potential academic issues early
- Warns about:
- Missing prerequisites
- Courses that don’t count toward the degree
- Graduation delays or risks
- Helps students make better decisions

# C. Transfer Recommendations

- Suggests CUNY colleges with strong programs in the student’s major
- Compares degree requirements across schools
- Shows which credits will transfer smoothly

# D. Cross-Campus Course Planning

- Uses the CUNY ePermit system
- Finds equivalent courses at other CUNY campuses
- Helps when classes are full, unavailable, or inconveniently scheduled

# E. Conversational AI Interface

- Allows students to ask questions in natural language
- Replaces the need to search through catalogs or documents
- Makes advising faster and easier to understand

# IV. Conclusion

- Helps students make smarter academic decisions
- Keeps students on track for graduation
- Reduces confusion by turning complex data into clear, personalized guidance
